# Team Task Manager - Frontend

A premium SaaS web application for team task management, built with React, Vite, TypeScript, and Tailwind CSS, based on the **Precision Architect** Stitch design.

## Features
- **Editorial Design**: High-density interface with tonal layering and minimal lines.
- **Role-Based Routing**: Dedicated views for Admins and Team Members.
- **Interactive Kanban**: Manage projects and tasks with drag-and-drop aesthetics.
- **Glassmorphism Navigation**: Floating sidebars with backdrop-blur.
- **Mock Data Ready**: Ready for demonstration with realistic data.

## Project Structure
- `src/components`: Reusable UI components (Button, Input, Modal, KanbanBoard).
- `src/layouts`: Shared application layouts (Dashboard, Auth).
- `src/pages`: Feature-specific pages (Admin, Member, Auth, Landing).
- `src/index.css`: Tailwind v4 theme configuration and design tokens.

## Getting Started
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Deployment
Prepared for deployment on **Railway** with `railway.json`.
